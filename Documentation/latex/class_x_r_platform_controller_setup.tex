\doxysubsection{Detailed Description}
Manages the setup of the VR controller models based on the active XR platform. This script enables or disables specific controller Game\+Objects depending on the XR loader being used. 

This class is designed to work with Unity\textquotesingle{}s XR system and specifically handles the Oculus VR platform. When the Oculus loader is active, it disables the default controller models and enables the Oculus-\/specific ones.
