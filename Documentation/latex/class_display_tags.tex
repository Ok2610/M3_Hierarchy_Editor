Updates the positions of the line between the parent and child transforms. This method is called when the parent or child moves, to ensure the line is updated accordingly. 

\texorpdfstring{$<$}{<}param name="{}"{}parent"{}"{}\texorpdfstring{$>$}{>}The parent transform, which is the start point of the line.

\texorpdfstring{$<$}{<}param name="{}"{}line\+Renderer"{}"{}\texorpdfstring{$>$}{>}The {\ttfamily Line\+Renderer} component that is responsible for drawing the line between the parent and child.


\begin{DoxyItemize}

\item The {\ttfamily Line\+Renderer} component\textquotesingle{}s positions are updated by setting the start (0) and end (1) points to the parent and child positions, respectively.
\item It is useful to call this method when either the parent or child transform moves, so that the line follows the new positions.
